## Usage Guidelines

- **Clear space**: Maintain minimum padding equal to the height of the speed lines
- **Minimum size**: 16px (use simplified favicon version at this size)
- **Dark backgrounds**: Use white logo
- **Light backgrounds**: Use black or blue logo
- **Don't**: Rotate, skew, add effects, or use colors outside the brand palette
